about=This plugin provides a simple interface to load paired images and their corresponding masks. It automatically matches images with their masks based on naming conventions (e.g., image.tif and image_mask.tif) and adds them to the QGIS project as grouped layers.

    Features:
    - Select separate directories for images and masks
    - Customizable mask suffix pattern
    - Automatic pairing of images with masks
    - Add selected or all pairs to map
    - Organize layers in groups
    - Clear all layers functionality
